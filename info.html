O projekte GitRepos

Moderný dashboard pre sledovanie trendov na GitHube.

Reálne dáta

Aplikácia sa pripája priamo na GitHub API a sťahuje vždy čerstvých 30 najpopulárnejších repozitárov sveta.

Inteligentný Filter

Automaticky detekujeme programovacie jazyky (Python, JS, Java...) a umožňujeme vám podľa nich filtrovať.

Triedenie

Zistite, čo je populárne. Zoraďte projekty podľa počtu hviezd, aktívnych vývojárov (forky) alebo chýb (issues).

Použité technológie

Tento projekt bol vytvorený ako školské zadanie. Je postavený na čistom HTML5 a CSS3 (využíva Grid a Flexbox pre responzivitu). Logiku zabezpečuje Vanilla JavaScript, ktorý asynchrónne komunikuje so servermi GitHubu. Font: Poppins.
